Spacings

The spacing between each images can be changed by using classes. ( spacing-0 / spacing-6 / spacing-10 / spacing-20 / spacing-30 )

spacing-0: 0px between gallery images

spacing-6: 6px between gallery images

spacing-10: 10px between gallery images

spacing-20: 20px between gallery images

spacing-30: 30px between gallery images

Hover Style 2

The hover style can be changed by adding hover-style-2 class to .gallery-wrapper element.
